# 🌍 MapApp — Интерактивная платформа для поиска и организации событий

## 1. Общая информация

Разрабатывается **интерактивная веб-платформа**, позволяющая пользователям находить и организовывать события в городе с удобным отображением на карте.

### 🔧 Технологический стек
- **Backend**: Python, FastAPI (асинхронный)
- **Frontend**: React / Vue / Angular (на выбор команды)
- **База данных**: PostgreSQL
- **Карта**: Яндекс.Карты API
- **Аутентификация**: JWT
- **Кэширование**: Redis
- **Фоновые задачи**: Celery
- **Чат в реальном времени**: через FastAPI

---

## 2. Регистрация и авторизация

- Регистрация через:
  - Имя, фамилия, возраст, email, пароль
- Создание профиля с историей событий
- Авторизация:
  - Логин/пароль

---

## 3. Карта (основа сайта) 🗺️

- Интеграция с **Яндекс.Картами**
- Отображение:
  - Заведений и общественных мест
  - Названия (и опционально — фото)
- Без оценок, комментариев и дополнительной информации (упрощённый режим)

---

## 4. Поиск и фильтрация 🔍

- Поиск по:
  - Названию заведения или события
  - Адресу
  - Категории
- Фильтры:
  - Время проведения
  - Тип отдыха (кино, спорт, кафе и т.д.)
  - Количество участников

---

## 5. События 🎉

Пользователь может создавать события:

- Выбор точки на карте или заведения
- Указание:
  - Название
  - Дата и время
  - Максимальное количество участников
  - Адрес
- Событие отображается на карте

---

## 6. Присоединение к событию ✅

- Просмотр деталей события:
  - Время, место, участники
  - Кнопка «Присоединиться»
  - Кнопка «Чат»
- После нажатия:
  - Пользователь добавляется в список участников
  - Событие появляется в его профиле

---

## 7. Профиль пользователя 👤

- Аватарка: загрузка
- Личные данные:
  - Имя
  - Фамилия
  - Email
  - Пароль
  - Возраст
- История событий:
  - Созданные
  - Посещённые

---

## 8. Чат внутри события 💬

- Базовый чат для участников события (MVP)
- Функционал:
  - Отправка сообщений
  - Получение в реальном времени
- Без файлов, эмодзи, истории (в MVP)

---

## 9. Архитектура MVP

| Компонент      | Технология                 |
|----------------|----------------------------|
| Backend        | FastAPI (Python), асинхронный |
| Frontend       | React / Vue / Angular      |
| База данных    | PostgreSQL                 |
| Кэширование    | Redis                      |
| Фоновые задачи | Celery                     |
| Аутентификация | JWT          |
| Карта          | Яндекс.Карты API           |
| Чат            | (FastAPI + frontend)       |

---

## 10. Требования к MVP

✅ **Должно быть реализовано:**
- Регистрация и авторизация
- Отображение карты
- Создание и поиск событий
- Присоединение к событию
- Профиль пользователя
- Чат в событии

❌ **Не реализовывать в MVP:**
- Оценки и рейтинги
- Комментарии
- Сложные фильтры (например, по интересам)
- Модерация контента
- Платные функции

---

> 🚀 Цель: быстро запустить рабочий прототип с базовым, но полезным функционалом.
